# Парсер
Запуск из main.py позволяет скачивать все обнаруженные проекты из Engee Gitlab, в которых находятся файлы формата `.engee`. 

Работает поэтапно:
- парсинг ссылок
- установка файлов

Все файлы (при запуске из `main.py`) сохраняются в папке `runs/`. Внутри создаются папки для каждого запущенного рана (с названием в формате `datetime-time` на момент запуска).

## Parser
Парсер собирает ссылки на проекты и файлы из проектов в тех репозиториях, где есть файлы `.engee`. Полученные файлы возвращает из функции main.
Сохраняет 2 файла: ссылки на проекты (caught_projects.txt) и ссылки на файлы (caught_models.txt). 

## Downloader
Скачивает файлы из гитлаба по переданным ссылкам, сохраняя их в папке `models/`.
